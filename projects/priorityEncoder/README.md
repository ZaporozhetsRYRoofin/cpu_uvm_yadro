# Priority Encoder (4-to-2)

## Описание
Модуль приоритетного шифратора на SystemVerilog.
Преобразует 4 входных бита в 2-битный код номера старшего активного бита.

## Принцип работы
- Если активен бит 3 → out = 11
- Если активен бит 2 → out = 10
- Если активен бит 1 → out = 01
- Если активен бит 0 → out = 00 
- Если все биты 0 → valid = 0

## Тестирование

Тест-бенч проверяет:
1. Все входы 0 → valid = 0
2. Каждый бит по отдельности
3. Приоритет при нескольких активных входах
4. Все биты активны одновременно
